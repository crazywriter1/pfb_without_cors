# pfb_without_cors
### Make sure you have git on your server

# Step by Step Intro

## 1. Step (Clone)

`git clone https://github.com/crazywriter1/pfb_without_cors `

## 2. Step (Install Packages)

`go mod tidy`

## 3. Step (Run)

`go run main.go`
